# 🤖 AI Chatbot Development

An AI-powered conversational chatbot built using **Python**, **Streamlit**, and **Large Language Models (LLMs)**. This project demonstrates the development of intelligent chatbots capable of maintaining contextual conversations, remembering previous interactions, and delivering accurate responses through prompt engineering and API integration.

---

## 🚀 Features

- 💬 Natural language conversations powered by LLMs
- 🧠 Session-based memory for contextual responses
- ⚡ Real-time AI-generated answers
- 🎯 Prompt engineering for improved response quality
- 🔗 AI API integration
- 📱 Interactive and responsive Streamlit interface
- ❌ Error handling and input validation
- 🌙 Clean and user-friendly UI

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/AI-Chatbot.git
```

Navigate into the project

```bash
cd AI-Chatbot
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
