# 🎓 GBU SupportGPT – AI Student Support Assistant

An AI-powered virtual assistant developed to simplify student support services at **Gautam Buddha University**. The chatbot provides instant assistance for common student queries related to the university portal, hostel facilities, library services, academics, scholarships, and other general concerns.

The goal of this project is to reduce response time, improve accessibility to information, and provide students with a centralized platform for university-related support.

---

## 🚀 Features

* 🎓 **Portal Support**

  * Login and registration assistance
  * Password reset guidance
  * ERP/Student Portal navigation
  * Profile update support

* 🏠 **Hostel Support**

  * Hostel registration process
  * Room allotment information
  * Fee details
  * Hostel rules and regulations
  * Complaint guidance

* 📚 **Library Support**

  * Book search assistance
  * Library timings
  * Borrowing and return policies
  * Digital library information
  * Membership support

* 📖 **Academic Support**

  * Course information
  * Semester registration guidance
  * Examination schedules
  * Attendance queries
  * Academic calendar assistance

* 💰 **Scholarship Support**

  * Scholarship eligibility
  * Required documentation
  * Application process
  * Important deadlines
  * Government and university scholarship information

* 💬 **General Support**

  * Frequently Asked Questions
  * Campus information
  * Contact details
  * Navigation assistance
  * Other student-related queries

## ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/GBU-SupportGPT.git
```

Navigate to the project directory

```bash
cd GBU-SupportGPT
```

Install the required dependencies

```bash
pip install -r requirements.txt
```

Run the application

```bash
streamlit run app.py
```
